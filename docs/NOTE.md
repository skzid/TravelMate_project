# Пояснительная записка

## Идея проекта

TravelMate — это небольшой дневник путешествий. Он помогает сохранять впечатления о городах, прикреплять фотографии и автоматически получать координаты места. 

## Реализация

Приложение написано на Flask. Основные маршруты находятся в файле `main.py`. Формы вынесены в `forms.py`, а модели базы данных — в папку `data`.

В проекте есть две ORM-модели:

- `User` — пользователь приложения;
- `Trip` — запись о месте или городе.


## Работа с пользователями

Регистрация и вход реализованы с помощью Flask-WTF и Flask-Login. Пароли не хранятся открытым текстом: перед сохранением они хешируются функцией `generate_password_hash`, а при входе проверяются через `check_password_hash`.

## Работа с файлами

При создании записи пользователь может загрузить фотографию. Файл сохраняется в папку `static/uploads`, а в базе данных хранится только имя файла. Разрешены изображения с расширениями `jpg`, `jpeg`, `png`, `gif`, `webp`.

## Внешний API

При добавлении города приложение обращается к Yandex Geocoder API и получает страну, долготу и широту. Если сеть недоступна, для нескольких городов используется локальный запасной словарь, чтобы проект можно было показать даже без интернета.

## REST API

В приложении реализованы маршруты:

- `GET /api/trips` — список публичных записей;
- `GET /api/trips/<id>` — одна публичная запись;
- `POST /api/trips` — создание записи из JSON.

Ошибки API возвращаются в формате JSON. Для превращения ORM-объектов в словари в модели `Trip` есть метод `to_dict`.

## Что можно улучшить

В дальнейшем можно добавить карту с отметками, поиск, фильтрацию по странам, комментарии и полноценный деплой на облачный хостинг.
